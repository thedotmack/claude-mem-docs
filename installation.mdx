## Common Questions

<AccordionGroup>
  <Accordion title="What just got installed?">
    - Hooks that capture your Claude sessions
    - MCP server for memory management
    - Local database for storing memories
    - Smart Trash for recovery
    - All in `~/.claude-mem/` (about 100MB)
  </Accordion>

  <Accordion title="Do I need to configure anything?">
    No. It works with zero configuration. Everything has sensible defaults.
  </Accordion>

  <Accordion title="What if something goes wrong?">
    ```bash
    claude-mem install --force
    ```
    This fixes 99% of issues.
  </Accordion>

  <Accordion title="How do I update?">
    ```bash
    npm update -g claude-mem
    claude-mem install --force
    ```
  </Accordion>

  <Accordion title="How do I uninstall?">
    ```bash
    claude-mem uninstall
    npm uninstall -g claude-mem
    ```
  </Accordion>
</AccordionGroup>
